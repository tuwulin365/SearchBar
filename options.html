Notes

Description:

This is displayed when you hover over a button.

Home page URL:

This determines the page you will be sent to if you perform a search without a query. Please include the "http" portion of the URL.

As with the Search URL, you can use %h to insert the host name of the current web page (e.g. for a WHOIS lookup), you can use %u to insert the full URL of the current web page (e.g. for an Internet Archive search), you can use the advanced %{<JavaScript expression>} syntax, and you can add "i:" to the start of a URL to force it to be opened in an incognito window.

Search URL:

SearchBar will usually be able to detect this automatically when you add a custom search via the right-click method. When this is not possible, you can try to obtain the search URL manually by performing a search using a test query, such as "TEST", copying the URL from the address bar, and replacing any instances of your test query with "%s".

As well as %s, you can also use %h to insert the host name of the current web page (e.g. for Google site search), and %u to insert the full URL of the current web page (e.g. for Google cache search).

For more advanced substitutions, you can also use %{<JavaScript expression>}. The expression will have access to the variables s, h and u as well as standard built-in JavaScript objects and functions such as JSON and encodeURIComponent. The expression will be evaluated using static-eval. For example, to insert the encoded search query but with spaces replaced with underscores, use %{encodeURIComponent(s.replace(/ /g,"_"))}.

For POST requests, you can use ?? in place of ?.

To force a URL to be opened in an incognito window, add "i:" to the start of the URL.

Icon URI:

You might like to use a website's favicon for its button icon. This is often linked to in a <link rel="shortcut icon"> or <link rel="icon"> tag near the top of the home page's source. It may be accessible from "https://www.example.com/favicon.ico" or "https://www.google.com/s2/favicons?domain=www.example.com", with "www.example.com" replaced by the website's host name.

Alternatively, you can use any image by using the "upload" feature to convert it to a data URI.

Hotkey:

Your primary search (highlighted at the top of the table) can, if it is enabled, be performed using the Enter key. You can also have another hotkey for this search.

Frequently asked questions

How do I add a particular search engine?

The easiest way to do this is to right-click on the search engine's input field and select "Add to SearchBar...". Alternatively, you can add custom searches manually via the advanced options under "Custom searches".

How do I open search results in a new tab?

Simply click the search button using the middle mouse button or holding the Ctrl key (just like for an ordinary link). To be able to open search results in a new tab by left-clicking, check the "Open search results in a new tab by default" checkbox.

You can also hold Ctrl while you are pressing a hotkey. Alternatively, you can specify your own new tab hotkeys via the advanced options under "Custom searches".

For your primary (leftmost) search (highlighted at the top of your custom searches), you can also press Ctrl+Enter.

How can I make SearchBar behave like a permanent toolbar?

Consider some of the following possibilities, available under "Advanced settings": enabling "Show SearchBar at the start of my browsing session"; disabling "Hide SearchBar when I go to a new page"; disabling "Show and hide SearchBar on each page independently"; and enabling "SearchBar displaces web pages".

Unfortunately, it is not possible to display SearchBar next to the address bar, since all Chrome extensions are restricted to a single toolbar button.

How can I backup or transfer my settings to another computer?

You can do this using either of the "Import/export settings" features under "Advanced settings". For the manual import/export, copy the text from the box and save it in a convenient location. Then, to restore your settings, paste the text into the box and click "Save changes". For Chrome sync, first make sure you are logged in to Chrome and have sync enabled. Then use the buttons to save and restore your settings to and from your Google account.

These features may not work if your settings take up too much space. The most common cause of this is large image uploads.

How do I perform two different searches at once?

You can do this by setting a hotkey for one custom search to be the same as a new tab hotkey for another custom search. This can be done via the advanced options under "Custom searches".

How can I highlight an exact phrase made up of more than one word?

Simply put the phrase inside quotation marks.

How do I get the hotkey for showing SearchBar to work on the New Tab page?

SearchBar's hotkeys may not function on the New Tab page or on Chrome settings pages. However, you can set another hotkey by selecting "Keyboard shortcuts" from the menu of Chrome's Extensions page.

Why does SearchBar sometimes appear in a popup when I press the toolbar button?

The full version of SearchBar uses content scripts and so cannot run on the New Tab page or on Chrome settings pages (such as this page). It will also not run on any hostnames you have specified under "Prevent SearchBar from running (except in a popup) on these hostnames", available under "Advanced settings". The "popup" version allows you to access the main features of SearchBar from these pages by using the toolbar button. However, features that interact with the web page, such as searching selected text and highlighting search terms, are disabled.

There is an option to always show SearchBar in a popup when the toolbar button is pressed, available under "Advanced settings". Occasionally the popup will appear even if you don't have this option selected and are on an ordinary web page on which you have allowed SearchBar to run. The most common reason for this is that the page has not yet finished loading.

Why does a small empty tooltip sometimes appear when I press the toolbar button?

The most common reason for this is that SearchBar is still loading. Loading times can be a little longer when you first open SearchBar. Please be patient.

Why do most "javascript:" URLs no longer work?

Previously, SearchBar allowed Home page URLs and Search URLs with the javascript: protocol. However, as of Manifest V3, Chrome extensions are no longer allowed execute arbitrary JavaScript supplied by the user. This is for improved security, as explained here. For this reason, Home page URLs and Search URLs with the javascript: protocol no longer work in SearchBar, except for a few special URLs, such as javascript:document.execCommand("copy"), which triggers the clipboard copy action.

Because of this restriction, SearchBar now supports an advanced %{<JavaScript expression>} syntax in Search URLs and Home Page URLs, such as in %{u.replace(/\/$/,"").replace(/\/[^\/]*$/,"").replace(/^[a-z-]*:\/$/,u)} and https://chrome.google.com/webstore/search/%{encodeURIComponent(s)}. This syntax is explained in more detail in the notes of the advanced options under "Custom searches".

Why does SearchBar need to "access my data on all websites"?

Chrome requires this permission to be asked for in order for SearchBar to be inserted onto every page. However, SearchBar does not change or use any of your data outside of its declared features. For more information, please see SearchBar's privacy policy, which can be found here.

How can I reset to the default settings?

You can do this using the "Manual import/export" feature under "Advanced settings" by deleting everything and clicking "Save changes".

What inspired you to create such an amazing extension?

This extension was borne out of personal frustration with Chrome's default search system and dissatisfaction with the features of similar extensions. I hope it improves your browsing experience as much as it has mine, and I would love to hear your feedback.
